\section{Requisitos de Hardware y Software}

\subsection{4.1 Tecnologías Seleccionadas}

\subsubsection*{Frontend}
\begin{itemize}
    \item \textbf{React:} Seleccionado por su robusto ecosistema y capacidad para crear interfaces dinámicas.
    \item \textbf{Mapas:} Uso de API como Google Maps, OpenStreetMap o Leaflet.
\end{itemize}

\subsubsection*{Backend}
\begin{itemize}
    \item \textbf{Kotlin:} Seleccionado por su seguridad de tipos y compatibilidad con Java.
    \item \textbf{Spring Boot:} Framework para desarrollo rápido y seguro.
    \item \textbf{PostgreSQL:} Base de datos elegida por:
    \begin{itemize}
        \item Manejo de grandes cantidades de datos.
        \item Buen rendimiento de consultas.
        \item Código abierto y gratuito.
        \item Herramientas para hacer copias de seguridad facilmente.
    \end{itemize}
\end{itemize}
    
    \item \textbf{Permitir que usuarios sin registrarse usen la app} pero solo podrían visualizar los incidentes, no podrían registrar un nuevo incidente ni actualizar el estado del incidente.
    
\end{itemize}
